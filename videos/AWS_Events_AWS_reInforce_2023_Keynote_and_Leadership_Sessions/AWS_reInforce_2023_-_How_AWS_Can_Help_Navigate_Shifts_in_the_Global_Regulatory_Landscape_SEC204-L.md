# AWS re:Inforce 2023 - How AWS Can Help Navigate Shifts in the Global Regulatory Landscape (SEC204-L)

[Video Link](https://www.youtube.com/watch?v=2Yu3QpmqTMs)

## Description

Russell Lewis, Principal Compliance Specialist, AWS for Financial Services, and Samara Moore, Senior Manager, Security Assurance, AWS, examine how the rapidly evolving regulatory landscape presents AWS and AWS customers with a unique and difficult compliance challenge. Regulatory engagement and assurance need to create the space for organizations to be more agile and innovative, while maintaining full compliance, satisfying regulators, and minimizing costs. In this session, hear from AWS leaders on how they are evolving as regulations change and helping customers meet these requirements.
