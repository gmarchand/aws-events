## Description

Securing a digital transformation or cloud adoption journey is often an exercise in tension as organizations search for their particular balance of innovation and risk while navigating the shared responsibility model. As external threats persist and the cloud threat landscape evolves, CISOs and other leaders are faced with re-evaluating the people, processes, and technology required to keep cloud environments safe. This lightning talk examines external threats as well as internal challenges for an organization and suggests practical advice for maturing cloud security posture, protection, detection, and response. Learn from the analysis of real-world cloud events and discover some hard-won lessons learned. This presentation is brought to you by SentinelOne, an AWS Partner.
