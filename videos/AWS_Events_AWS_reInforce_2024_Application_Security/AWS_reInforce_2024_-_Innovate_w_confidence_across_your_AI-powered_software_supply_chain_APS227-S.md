## Description

As developers adopt generative AI tooling, organizations must grapple with how to secure those generative AI workflows to ensure a thorough approach to code security. In a recent survey targeting hundreds of engineering and security professionals, Snyk found that 96% of teams use generative AI coding tools, making them part of the software supply chain, but 80% of those developers will bypass AI code security policies. In this lightning talk, learn some of the key findings from Snyk’s “2023 AI Code Security Report,” and learn best practices on security measures that teams can take to manage the supercharged pace of software development, securely. This presentation is brought to you by Snyk, an AWS Partner.
