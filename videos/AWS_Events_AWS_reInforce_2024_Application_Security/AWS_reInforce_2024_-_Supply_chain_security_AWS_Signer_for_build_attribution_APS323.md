# AWS re:Inforce 2024 - Supply chain security: AWS Signer for build attribution (APS323)

[Video Link](https://www.youtube.com/watch?v=jGKfZmPCDZk)

## Description

In this talk, a multinational communications technology company discusses how they are using AWS Signer as part of a holistic approach to shifting security left in a build pipeline. As they move more of their security tooling into automated processes, it becomes increasingly important to properly attribute those artifacts back to where they were created and verified. AWS Signer adds a layer of defense to help ensure the deployed artifacts are sourced properly to help protect against supply chain attacks.
