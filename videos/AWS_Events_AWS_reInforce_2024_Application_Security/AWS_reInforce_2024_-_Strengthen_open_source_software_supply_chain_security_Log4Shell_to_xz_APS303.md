## Description

Open source software has become ubiquitous, adding value at every stage in software development. However, recent vulnerabilities have highlighted the importance of open source supply chain security across the technical world. Developers, security responders, and regulators are grappling with this challenge daily. In this session, dive deep into lessons learned from navigating past supply chain crises—and some that almost were. Leave this session equipped with actionable techniques and best practices to improve the security posture of your supply chain.
