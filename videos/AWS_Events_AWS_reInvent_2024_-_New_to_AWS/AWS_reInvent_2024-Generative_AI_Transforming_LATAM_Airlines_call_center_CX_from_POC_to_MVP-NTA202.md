# AWS re:Invent 2024-Generative AI: Transforming LATAM Airlines' call center CX from POC to MVP-NTA202

[Video Link](https://www.youtube.com/watch?v=DYfT9XDUUQk)

## Description

LATAM Airlines transformed their call center using generative AI via Amazon Bedrock, moving from reactive to proactive customer service. This session explores their journey from proof of concept to MVP, highlighting improved intent categorization for automated handling or agent escalation. Key insights include AI training requirements, architectures, and achieving 85% response accuracy. Learn about integrating Genesys Cloud with AWS services and LATAM’s progression from handling 70,000 calls in the Peruvian market POC to an anticipated 1.5 million monthly calls with their MVP solution. Discover how generative AI is revolutionizing call center operations and enhancing customer experience.
