## Description

Searching online for answers to technical questions can be frustrating, especially when you need an accurate response. That’s why AWS created AWS re:Post, a new community-driven question-and-answer service to help cloud professionals quickly remove technical roadblocks. AWS re:Post lets you leverage the expertise of a vibrant community of cloud enthusiasts, including AWS users, partners, and employees. Browse AWS vetted Q&A pairs across all AWS technology domains, or log in and start asking and answering questions to build your “community expert” status. Join this session for an overview and in-depth demonstration of AWS re:Post.
