# AWS re:Invent 2025 - Modern Secrets: A Journey from Legacy Systems to AI-Ready Security (SEC213)

[Video Link](https://www.youtube.com/watch?v=FMxP-jvrd6U)

## Description

Learn from Acquia's journey from complex legacy systems to cutting-edge secrets management with AWS Secrets Manager. Discover how Acquia revolutionized its security infrastructure, mastering multi-tenant isolation and achieving zero-downtime migration across hundreds of services. This is a potential blueprint for your modern security architecture. Perfect for forward-thinking architects and security leaders, you'll get exclusive insights into Acquia's transformation and how Acquia leveraged its relationship with AWS to drive its roadmap for Agentic AI integration and improved security via automated secrets rotation. This is valuable glimpse into large-scale and seamless secrets management.
