# AWS re:Invent 2025 - Fidelity: AWS Health & Support Data Pipelines to GenAI Actions (SPS318)

[Video Link](https://www.youtube.com/watch?v=tWULplELhxI)

## Description

Transform high-volume AWS Health and Support data into strategic advantage. While AWS provides comprehensive data streams, the volume and complexity present challenges for teams managing thousands of applications. This session explores how Fidelity collaborated with AWS through CENTS (Cloud Events Notification Transport System), successfully ingesting AWS Health data using prescriptive guidance. We demonstrate event routing across thousands of accounts, then showcase adding Support data and Amazon Bedrock for intelligent automation. Learn how GenAI enables individualized guidance and aggregate synthesis, advancing from reactive monitoring to predictive AI-augmented operational excellence with proven strategies.
