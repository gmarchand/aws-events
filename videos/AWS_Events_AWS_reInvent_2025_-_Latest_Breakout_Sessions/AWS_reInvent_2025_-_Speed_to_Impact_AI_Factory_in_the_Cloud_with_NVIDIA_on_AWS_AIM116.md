# AWS re:Invent 2025 - Speed to Impact: AI Factory in the Cloud with NVIDIA on AWS (AIM116)

[Video Link](https://www.youtube.com/watch?v=pjLYOw17bPA)

## Description

Developing and deploying mission-critical AI systems is increasingly complex and resource-intensive. This panel discussion demonstrates how organizations can accelerate AI development with turnkey, full-stack AI factories in the cloud—combining NVIDIA’s optimized hardware, software, and AI expertise with AWS’ scalable, co-engineered infrastructure. Attendees will hear customer success stories illustrating how this collaboration enables rapid, efficient delivery of advanced generative and Agentic AI applications by optimizing every infrastructure layer and leveraging best-in-class tools from both NVIDIA and AWS. This presentation is brought to you by NVIDIA, an AWS Partner.
