# AWS re:Invent 2025 - Build Advanced Search with Vector, Hybrid, and AI Techniques (ANT314)

## Description

Amazon OpenSearch Service is at the heart of how Artificial Intelligence is transforming the way we store, search, and use data. Learn how businesses are using AI to power their search-backed applications, achieving significant improvements in search accuracy and performance with OpenSearch Service’s advanced vector capabilities. Learn how Recruit uses OpenSearch Service to find available tables for "Hot Pepper Gourmet" diners. Dive deep into OpenSearch’s vector capabilities, and its role in building chat bots and agentic search. You’ll learn how to build scalable, high-performance, and cost-optimal solutions so that you can incorporate these technologies in your stack.
