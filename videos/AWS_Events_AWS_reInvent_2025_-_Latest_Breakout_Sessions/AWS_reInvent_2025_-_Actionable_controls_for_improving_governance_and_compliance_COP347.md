# AWS re:Invent 2025 - Actionable controls for improving governance and compliance (COP347)

[Video Link](https://www.youtube.com/watch?v=2bJ9F0tr29s)

## Description

Organizations often struggle to translate complex regulatory requirements into practical controls while maintaining continuous compliance and security assurance. Learn how to transform compliance frameworks into actionable AWS controls through effective gap analysis and control mapping. This session shows how to leverage AWS Control Tower, AWS Security Hub, AWS Config, and AWS Audit Manager to build and maintain a scalable governance strategy. We'll explore real-world examples of mapping common controls across multiple frameworks, implementing automated compliance checks, and creating custom control deployments. Discover how to streamline compliance reporting while maintaining continuous security assurance across your AWS organization.
