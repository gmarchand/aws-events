# AWS re:Invent 2025 - Building Prudential’s microagent platform with MCP and A2A on AWS (IND3302)

[Video Link](https://www.youtube.com/watch?v=9UTzSY40e9I)

## Description

Enterprises are exploring ways to scale their AI agent implementations beyond initial use cases. Learn how Prudential Financial is moving beyond single-purpose deployments with a scalable, microagent architecture powered by Amazon Bedrock and MCP agent-to-agent (A2A) communication. Prudential will share how their financial advisor assistant became the foundation for an enterprise multi-agent system, supporting diverse scenarios through modular design, independent scaling, and seamless deployment on AWS. Whether you’re exploring AI agents for specific business needs or building a foundation for enterprise-wide adoption, walk away with patterns and best practices for scaling AI agents.
