# AWS re:Invent 2025 - Modernize Apple platform development with AWS and EC2 Mac (CMP334)

[Video Link](https://www.youtube.com/watch?v=uFIRieQbPvg)

## Description

EC2 Mac instances allow you to run on-demand macOS workloads in the cloud and extend the flexibility, scalability, and cost benefits of AWS to all Apple developers. In this session, you will learn about Apple application development in the cloud using EC2 Mac instances. We will share the latest features and advancements for EC2 Mac and how customers are benefiting from them. You will also hear firsthand from a customer on how they have optmized their Apple development worklfow and benefited from Apple application development in the cloud.
