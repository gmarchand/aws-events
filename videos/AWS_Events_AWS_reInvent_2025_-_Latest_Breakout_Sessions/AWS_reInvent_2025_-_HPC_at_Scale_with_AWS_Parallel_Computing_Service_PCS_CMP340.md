# AWS re:Invent 2025 - HPC at Scale with AWS Parallel Computing Service (PCS) (CMP340)

[Video Link](https://www.youtube.com/watch?v=XjM0VtDDbt4)

## Description

High performance computing workloads are growing in complexity, scale, and importance. AWS Parallel Computing Service (PCS) brings simplicity and flexibility to this challenge by automating cluster creation, job scheduling, and resource management. In this session, learn how PCS helps organizations move faster, scale securely, and achieve predictable performance for demanding workloads in energy, manufacturing, and research. Customers will share their experience building and running real-world HPC environments with PCS, including lessons learned and best practices. Join us to see how PCS is redefining the way high performance computing runs on AWS.
