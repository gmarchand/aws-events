# AWS re:Invent 2025 - Multicloud security best practices (HMC318)

[Video Link](https://www.youtube.com/watch?v=v4JwkLsjyx0)

## Description

Organizations operating in multicloud environments face challenges with distributed data security, identity management, and continuous monitoring. In this session, we will explore approaches to address multicloud security, aligned with NIST Cybersecurity Framework (CSF), an industry standard, focused on protect, detect, and respond functions. We will dive deep into security visibility via centralized logging, encryption and privacy enforcement, and centralized federation. You will learn specific architectural patterns that will help address multicloud security and identity challenges.
