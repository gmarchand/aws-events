# AWS re:Invent 2025 -Stripe: Revolutionizing compliance workflows w/ agentic infrastructure (IND3300)

[Video Link](https://www.youtube.com/watch?v=NKF4qKnOaCE)

## Description

Financial services companies struggle to scale Enhanced Due Diligence (EDD) processes while maintaining quality as merchant volumes grow. Learn how Stripe revolutionized compliance operations by building Eddie (EDD Detective)—an AI agent built on Amazon Bedrock that automates complex merchant risk assessments, reducing case handling time by 70% and enabling analysts to review 3x more cases. Discover how they built reliability and observability into agent operations, handling thousands of investigations daily while ensuring accuracy of AI-generated insights. Explore architectural patterns for orchestrating multi-step workflows, implementing human-in-the-loop validation, and maintaining audit trails at scale.
