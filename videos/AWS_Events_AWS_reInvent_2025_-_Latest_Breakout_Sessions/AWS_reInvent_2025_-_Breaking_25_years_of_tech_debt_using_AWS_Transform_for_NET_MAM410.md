# AWS re:Invent 2025 - Breaking 25 years of tech debt using AWS Transform for .NET (MAM410)

[Video Link](https://www.youtube.com/watch?v=JF1KJkSpFeM)

## Description

Discover how organizations are accelerating modernization of their legacy .NET Framework applications by up to 4x using AWS Transform. This session explores how Idemia, a leading identity solutions provider in the US, addressed 25 years of technical debt by using AWS Transform for modernizing their legacy .NET Framework applications to cross-platform .NET. We'll showcase AWS Transform's latest .NET features and share practical insights on maintaining business continuity during modernization of mission-critical applications.
