# AWS re:Invent 2025 - Building a Secure Foundation for SAP RISE and Non-SAP Workload (NTA318)

[Video Link](https://www.youtube.com/watch?v=Q293c8Kj5Mk)

## Description

Discover how global enterprises successfully migrate SAP RISE and non-SAP workloads to AWS. This session explores the critical foundational aspects of AWS cloud migration, showcasing the implementation of a multi-account strategy that optimizes security and resource management.You'll learn how to build a secure landing zone, set up comprehensive backups, and establish integrated Security Information and Event Management (SIEM) capabilities.Learn how these key elements formed the backbone of AWS cloud migration, supporting both SAP and non-SAP ecosystems. You'll gain insights into overcoming challenges and implementing strategies that can accelerate AWS cloud adoption.
