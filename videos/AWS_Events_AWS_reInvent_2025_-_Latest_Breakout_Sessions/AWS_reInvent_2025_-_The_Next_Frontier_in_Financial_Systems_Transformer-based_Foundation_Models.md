# AWS re:Invent 2025 - The Next Frontier in Financial Systems: Transformer-based Foundation Models

[Video Link](https://www.youtube.com/watch?v=5aLPLAx72Sg)

## Description

Transformer-based foundation models are revolutionizing payment systems by providing deep contextual understanding of sequential transaction data. In this session, NVIDIA and AWS experts explain how large-scale tabular foundation models are trained and deployed to detect fraud in real time, personalize customer experiences, and interpret complex transaction patterns. Attendees will learn about the cloud-based AI factories enabling scalable deployment, NVIDIA’s accelerated computing for efficient training and tuning, and AWS’s secure and scalable cloud infrastructure. This presentation is brought to you by NVIDIA, an AWS Partner.
