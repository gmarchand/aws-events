# AWS re:Invent 2025 - Secure AI Agents Delivering Efficiency in Regulated Industries (WPS317)

## Description

Regulated industries demand more than AI deployment—they require secure, efficient, and measurable outcomes. Holland Casino modernized cloud operations with a multi-agent system using AWS Strands Agents SDK, orchestrating agents for security, cost optimization, and operations support with fine-grained permissions. Sicoob, Brazil’s largest credit union, transformed operations with AI agents and digital employees: achieving 80%+ case resolution, cutting document processing time by 25%, deploying 80 digital employees to save 440,000 hours and R$23M, and automating 85% of credit operations. In this session, you’ll learn how organizations securely scale AI to cut costs, streamline compliance, and modernize operations.
