# AWS re:Invent 2025 - Building and managing conversational AI at scale: lessons from Alexa+ (AMZ305)

[Video Link](https://www.youtube.com/watch?v=HBzxf6-7GhQ)

## Description

This technical session dives deep into building production-grade conversational AI systems at scale through the lens of Alexa+'s architecture. Learn how to implement real-time inference, manage strict latency requirements, and deploy AI guardrails using AWS services like using a comprehensive suite of AWS services including Amazon Bedrock, Amazon SageMaker, Amazon Lambda, Amazon ECS, Amazon DynamoDB, and CloudWatch. We'll explore architecture patterns for processing millions of conversations daily while maintaining high accuracy and safety, demonstrating how to apply Well-Architected principles to build robust, scalable conversational AI systems.
