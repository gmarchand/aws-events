# AWS re:Invent 2025-From fragmented to unified platform: Rocket Companies data and AI journey-IND3306

[Video Link](https://www.youtube.com/watch?v=nfVRNsFIfBA)

## Description

Enterprises seek to unify data access across the organization, safeguard sensitive client information, and scale MLOps efficiently. Learn how Rocket Companies addressed these challenges by building a unified data and AI platform on AWS. Discover how Rocket leveraged Amazon SageMaker for machine learning workflows, integrated with custom Amazon EKS applications and CI/CD pipelines, and implemented MLflow for model governance. See how Amazon DataZone can support secure enterprise-wide metadata management. Join this session to explore architectural patterns for data democratization, combined with MLOps excellence, showcasing practical solutions that streamline data and machine learning workflows while maintaining security controls.
