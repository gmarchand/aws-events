# AWS re:Invent 2025 - Accelerating Development and DevSecOps with Amazon Bedrock and Kiro (PEX303)

[Video Link](https://www.youtube.com/watch?v=P2YX43I8GFI)

## Description

Discover how AWS partners are leveraging Amazon Bedrock and Kiro to revolutionize both software architecture design and DevSecOps practices. Through real-world examples, explore powerful patterns that enhance the entire development lifecycle. Learn how to use GenAI to accelerate cloud architecture design processes—generating standardized patterns, optimizing configurations, ensuring compliance, and creating documentation. Explore how to integrate AI capabilities with security scanning tools, automated code generation, CI/CD pipelines, and threat modeling frameworks. This session showcases how AWS partners have implemented these solutions to improve consistency, maintain compliance, and automate security tasks. Bring your questions to this collaborative session.
