# AWS re:Invent 2025 - Making Level 4 Autonomous Networks a reality with British Telecom (IND205)

[Video Link](https://www.youtube.com/watch?v=QYAYR0LmO8U)

## Description

BT is partnering with AWS to create a fully autonomous "Dark NOC" by integrating AI/ML, generative AI, and Agentic AI. Moving from traditional telecom to software-driven operations, BT has implemented GenAI-driven AIOps solutions that reduce costs and improve customer experience. The system uses AWS's agentic AI with autonomous agents and Amazon Sagemaker ML models to enable intelligent monitoring, automated network management, and advanced diagnostics. These AI agents proactively handle network issues and identify customer impacts, advancing BT's goal of zero-touch network operations.
