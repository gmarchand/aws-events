# AWS re:Invent 2025 - Building Scalable, Self-Orchestrating AI Workflows with A2A and MCP (DEV415)

[Video Link](https://www.youtube.com/watch?v=9O9zZ1lQWiI)

## Description

Learn how to build scalable, autonomous AI workflows using the Agent-to-Agent (A2A) protocol and Model Context Protocol (MCP). This session demonstrates how to design specialized agents that dynamically discover each other's capabilities and collaborate on complex tasks across platforms and implementations. Using MCP for standardized context integration, you'll build horizontally scalable, serverless agents with AWS Lambda, Amazon Bedrock, and Momento Cache. You'll also learn how A2A enables secure agent collaboration with built-in observability. Through real-world examples and best practices, you'll leave equipped to design, deploy, and scale interoperable AI systems in production.
