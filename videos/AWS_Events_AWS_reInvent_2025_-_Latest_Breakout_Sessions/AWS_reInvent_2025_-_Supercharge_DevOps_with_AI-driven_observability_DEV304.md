# AWS re:Invent 2025 - Supercharge DevOps with AI-driven observability (DEV304)

[Video Link](https://www.youtube.com/watch?v=TsKKTVepP7k)

## Description

Modern observability extends beyond logs and metrics to deliver understanding, context, and prediction. This session demonstrates how Generative AI transforms DevOps and SRE practices in CI/CD environments. Learn to build AI-enhanced observability systems using Amazon Bedrock and Amazon Q that automatically explain failures, correlate events, and provide predictive recommendations. Explore techniques for creating AI agents that simulate scenarios and stress-test architectures, enabling proactive insights with clear traceability. Attendees will gain practical knowledge to enhance their automation and incident response capabilities through AI-driven observability.
