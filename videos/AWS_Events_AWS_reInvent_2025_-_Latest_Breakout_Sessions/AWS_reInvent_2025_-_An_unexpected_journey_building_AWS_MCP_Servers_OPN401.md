# AWS re:Invent 2025 - An unexpected journey building AWS MCP Servers (OPN401)

[Video Link](https://www.youtube.com/watch?v=kSX1utwig9k)

## Description

This session explores the evolution from generic AI coding assistants to AWS-intelligent development tools through the Model Context Protocol (MCP). Attendees will learn how MCP transforms AI assistants by injecting AWS-specific context, best practices, and service knowledge directly into development workflows. We'll demonstrate using 25+ open-source AWS MCP servers that provide real-time access to CDK patterns, cost analysis, Amazon Bedrock capabilities, and infrastructure guidance. Through live demos and examples, you'll discover how to configure and leverage existing MCP servers that understand AWS services. Leave with practical skills to enhance your AI assistants for accelerated serverless and AI application development.
