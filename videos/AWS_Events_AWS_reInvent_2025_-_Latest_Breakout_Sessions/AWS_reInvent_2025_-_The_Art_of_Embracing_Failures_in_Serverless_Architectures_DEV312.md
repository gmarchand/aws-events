# AWS re:Invent 2025 - The Art of Embracing Failures in Serverless Architectures (DEV312)

## Description

Serverless architectures offer the immense power of distributed systems wrapped in irresistible simplicity. But that simplicity is an illusion. Explore hidden pitfalls in distributed systems and practical strategies to build resilience, illustrated with real-world examples using AWS Lambda and Amazon Kinesis. Above all, embrace failures. In the words of Dr. Werner Vogels: everything fails, all the time.
