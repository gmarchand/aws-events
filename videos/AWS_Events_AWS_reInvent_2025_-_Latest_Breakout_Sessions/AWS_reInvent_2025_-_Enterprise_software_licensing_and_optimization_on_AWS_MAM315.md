# AWS re:Invent 2025 - Enterprise software licensing and optimization on AWS (MAM315)

[Video Link](https://www.youtube.com/watch?v=4HSY9Cc3U1w)

## Description

This breakout session dives into licensing commercial software on AWS, with a focus on Microsoft workloads. Join our Microsoft Optimization and Licensing Assessment (OLA) specialists to explore the best practices for navigating licensing complexities in the cloud while maximizing cost-efficiency. Learn about fully-funded OLA offerings for on-premises workloads, cloud to AWS migrations, existing EC2 deployments, and the new self-serve AWS Transform Assessments. Discover how these assessments provide valuable insights and right-sizing recommendations across multiple services such as Elastic Compute Cloud (EC2).
