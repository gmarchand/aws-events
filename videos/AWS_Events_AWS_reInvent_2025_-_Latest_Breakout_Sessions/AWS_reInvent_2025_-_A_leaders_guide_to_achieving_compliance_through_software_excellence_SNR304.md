# AWS re:Invent 2025 - A leader's guide to achieving compliance through software excellence (SNR304)

[Video Link](https://www.youtube.com/watch?v=uWLaSR6YqjA)

## Description

Traditional software development often resembles a manufacturing assembly line, where code passes from one developer to the next, creating idle time and fragmented documentation that fails to meet modern compliance demands. Modern application development offers a better way to simultaneously enhance productivity and satisfy regulatory requirements. Through integrated tooling and practices—including Git version control and automated testing—organizations can maximize developer productivity while naturally producing documentary proof that serves as compliance evidence. This session features real-world examples from leaders who have successfully leveraged these practices to meet critical compliance standards (FDA 21 CFR Part 11, ISO 13485, GMP Annex 11).
