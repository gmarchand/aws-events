# AWS re:Invent 2025 - Scale modernization with agentic AI featuring Thomson Reuters (MAM334)

[Video Link](https://www.youtube.com/watch?v=L5oam0A1CDw)

## Description

Join Thomson Reuters to discover how they leveraged agentic AI service to revolutionize their .NET modernization journey. In this session you will learn how they leveraged AWS Transform for .NET to modernize 5M lines of code monthly, achieving a 4x acceleration in development velocity. Learn how they reduced application transformation time from months to a single sprint while cutting operating costs by 30%. This session demonstrates practical implementation of parallel modernization at scale, featuring both web and IDE experiences, and shares insights on using AI-powered agents to dramatically reduce technical debt while maintaining security and performance.
