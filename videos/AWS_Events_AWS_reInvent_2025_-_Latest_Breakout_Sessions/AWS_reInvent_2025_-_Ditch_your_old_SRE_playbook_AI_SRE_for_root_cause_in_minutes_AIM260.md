# AWS re:Invent 2025 - Ditch your old SRE playbook: AI SRE for root cause in minutes (AIM260)

[Video Link](https://www.youtube.com/watch?v=YrdtOwMzJCU)

## Description

When the pager goes off, engineers face a wall of noise: logs, metrics, traces, configs, code, and fractured tribal knowledge. Traditional observability adds data but rarely a path to resolution, leaving teams with heavy cognitive load and little context of the whole production system. In this session, see how AI SRE agents cut through the chaos by reasoning across every source, using tools like humans, testing hypotheses, surfacing evidence-backed root causes in minutes, and learning from every interaction. Learn what it takes to make multi-agent AI SRE production-ready, and how enterprises use Resolve AI to put machines on-call for humans. This presentation is brought to you by Resolve AI, an AWS Partner.
