# AWS re:Invent 2025 - Simplify connectivity from remote sites - updates and eero integration (NET337)

[Video Link](https://www.youtube.com/watch?v=bt5IwPVnSnw)

## Description

Learn how AWS is simplifying enterprise connectivity for remote sites and distributed workforces. In this session, dive deep into latest innovations in AWS Site-to-Site VPN, showcasing how enterprises can establish secure and reliable connections from remote locations to their AWS workloads. Learn about the exciting collaboration between eero and AWS that's transforming remote site management with one-click integrations into services like AWS Site-to-Site VPN, AWS Transit Gateway, and AWS Cloud WAN. Discover how this solution delivers a centralized, intuitive management interface that eliminates the complexity of traditional network infrastructure while maintaining enterprise-grade security and performance.
