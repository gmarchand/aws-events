# AWS re:Invent 2025 - Secure in milliseconds: Visa's AI-powered fraud defense on AWS (IND3313)

[Video Link](https://www.youtube.com/watch?v=Ms0lDo_19sM)

## Description

Discover how Visa revolutionized real-time payment security by leveraging AWS services, including AWS Nitro Enclaves, Amazon EKS, and Amazon MemoryDB, for their Visa Protect solution. This session explores Visa's journey to a rapidly scalable, cloud-based fraud detection solution. Learn how Visa processes massive volumes of sensitive financial data with ultra-low latency while maintaining stringent security standards. We'll dive into the multi-region architecture, real-time AI/ML fraud detection on Amazon EKS, and compute-level isolation that allowed Visa to achieve sub-250 millisecond fraud detection latency and 99.99% availability. Join us to uncover the blueprint for secure, AI-driven payment risk scoring.
