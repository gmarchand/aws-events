# AWS re:Invent 2025 - Learn How Sky Powers the Future of Streaming with Redis Cloud (API209)

[Video Link](https://www.youtube.com/watch?v=tlNxhm9LF4U)

## Description

When Sky streamed to 23 million viewers during the Peacock-exclusive NFL Wild Card game, bookmarking ensured a seamless experience, handling up to 140,000+ writes per second during live events and 116,000 during record-breaking moments. To meet these demands, Redis Cloud delivered: cross-region active-active replication for global sync, sub-millisecond latency for spikes, and intelligent auto-scaling. Its write-behind capabilities cut costs and enabled a long-awaited AWS Keyspaces migration. Now, Sky’s GenAI platform, Genapi, leverages Redis Cloud's vector search and semantic caching to transform content discovery. Redis Cloud isn’t just solving scale, it’s powering Sky’s AI-driven future. This presentation is brought to you by Redis, an AWS Partner.
