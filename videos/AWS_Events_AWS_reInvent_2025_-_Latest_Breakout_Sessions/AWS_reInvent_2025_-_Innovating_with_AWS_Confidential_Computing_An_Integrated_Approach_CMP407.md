# AWS re:Invent 2025 - Innovating with AWS Confidential Computing: An Integrated Approach (CMP407)

[Video Link](https://www.youtube.com/watch?v=R2QxpJDEmY4)

## Description

Confidential computing enables customers to protect code and data from unauthorized access during processing.  This session explores AWS's innovation on confidential computing capabilities including the AWS Nitro System, AWS Nitro Enclaves, and the Nitro Trusted Platform Module.  Learn how Nitro System capabilities can be used to meet evolving data protection challenges, such as confidential inferencing. The session will cover concepts that range from isolation from cloud operators to leveraging cryptographic capabilities on CPU, GPU, and AI accelerator-based instances for applications handling highly sensitive data such as personally identifiable information, financial assets, medical information, intellectual property, and AI data.
