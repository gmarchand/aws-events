# AWS re:Invent 2025 - From Static to Dynamic: Fortive’s Next-Gen AI Platform on AWS (MAM356)

[Video Link](https://www.youtube.com/watch?v=L6J5TsZcPt4)

## Description

The Fortive Brain Agentic AI Proof of Concept (POC) on AWS transitioned Fortive’s static Azure-based AI platform into a dynamic, scalable, and centralized AWS solution. Leveraging Amazon Bedrock and cloud-native technologies, it streamlined OpCo access to AI services, enabled real-time standardized data management, and seamless integration with various data sources. Over 10 weeks, the project automated infrastructure deployment, integrated GitHub and Postgres SQL, and built conversational AI assistants. Key results included reduced research time, faster development, improved onboarding, cost savings, and enhanced business agility, empowering future-ready enterprise-wide AI adoption.
