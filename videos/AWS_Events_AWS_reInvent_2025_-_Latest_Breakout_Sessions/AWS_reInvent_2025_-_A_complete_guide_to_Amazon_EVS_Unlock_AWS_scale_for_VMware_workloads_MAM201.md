# AWS re:Invent 2025 - A complete guide to Amazon EVS: Unlock AWS scale for VMware workloads (MAM201)

[Video Link](https://www.youtube.com/watch?v=d0TLechcV74)

## Description

Amazon Elastic VMware Service (Amazon EVS) is your fastest path to run VMware workloads on AWS, eliminating the need to replatform or refactor. Discover how Amazon EVS helps you unlock AWS scale and agility while maximizing existing VMware investments and skillsets. Explore core building blocks and guidance on self-managing or leveraging AWS Partners for your environments. We'll cover strategies for customizing your virtualization stack in the cloud, including integration with storage, backup, and disaster recovery solutions. Through a live demo, we'll showcase intuitive deployment workflows and share the latest service innovations to provide you with more choice and flexibility.
