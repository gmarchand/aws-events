# AWS re:Invent 2025 - Building Zero-Trust AI Agents with Okta, Auth0 and AWS (AIM249)

[Video Link](https://www.youtube.com/watch?v=qVp7pGuP4LI)

## Description

This session unveils Auth0 for AI Agents—a powerful solution enabling humans and non-human identities (NHI) to operate securely across AWS applications such as Bedrock and Quick Suite. Discover how Okta and Auth0 combine to empower agents with granular access controls and better user experiences. Through practical demos and real-world scenarios, we'll showcase how Auth0 for AI Agents leverages XAA (Cross App Access) and Model Context Protocol (MCP) to ensure secure collaboration within and across your AWS applications. Learn how to unlock the full potential of your agents with simplified identity management, robust security, organizational policies and user consent. This presentation is brought to you by Okta, an AWS Partner.
