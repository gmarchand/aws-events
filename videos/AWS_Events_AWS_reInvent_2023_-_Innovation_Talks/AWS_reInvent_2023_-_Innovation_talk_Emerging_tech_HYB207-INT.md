## Description

Companies today are creating some of the most exciting and cutting-edge innovations with AWS. This talk highlights some of the unique ways companies are using AWS—including for hybrid environments, edge computing, high-performance computing, quantum computing, the Internet of Things, and beyond. Learn about some of the most dynamic use cases, like genomics and drug discovery; 5G networks; smart factories; uses in unconditioned environments like construction sites and oil platforms; and uses in mobile environments like ships, airplanes, and space stations.
