## Description

How can you turn your existing data architecture like data lakes into a business advantage with generative AI? In this talk, explore what is top of mind for leaders of data organizations across the globe for how they can best put proprietary datasets to work for unique, differentiated generative AI solutions. Learn from AWS customers with production generative AI applications about how data has factored into pretraining foundation models and customization through techniques like Retrieval Augmented Generation (RAG). Join this talk to understand three key data considerations for any usage of generative AI applications and how AWS is working backwards from the customer to help solve these challenges.
