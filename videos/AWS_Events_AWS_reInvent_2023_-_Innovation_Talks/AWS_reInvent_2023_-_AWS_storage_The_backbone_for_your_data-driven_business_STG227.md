# AWS re:Invent 2023 - AWS storage: The backbone for your data-driven business (STG227)

[Video Link](https://www.youtube.com/watch?v=Alxig9GFIE4)

## Description

AWS offers reliable, scalable, and secure storage services to power your applications, analytics, and AI/ML workloads. In this Innovation Talk, Andy Warfield, AWS Distinguished Engineer, shares the latest storage innovations from AWS and inside perspectives on how to build a resilient, agile data foundation that accelerates insights and innovations. Discover how organizations are driving a competitive advantage by enabling faster data access, simplifying data lakes, and supercharging AI/ML initiatives using high-performance storage from AWS. Andy provides key takeaways about how AWS storage acts as the backbone for your data-driven business—supporting security, governance, analytics, and application development.
