# AWS re:Invent 2023 - Improve FMs with Amazon SageMaker human-in-the-loop capabilities (AIM334)

[Video Link](https://www.youtube.com/watch?v=9T775vCL-ak)

## Description

Amazon SageMaker Ground Truth offers a set of human-in-the-loop capabilities for machine learning, allowing you to harness the power of human feedback across the ML lifecycle. In this session, learn how AWS uses human input to help you create high-quality training datasets for your generative AI applications (through supervised fine tuning and reinforcement learning with human feedback) and customize foundation models (FMs) on specific tasks or with company- and domain-specific data. Join this session to see real-world examples and demos.
