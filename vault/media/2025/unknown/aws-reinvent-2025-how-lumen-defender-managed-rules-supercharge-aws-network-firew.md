# How Lumen Defender Managed Rules Supercharge AWS Network Firewall

## Session Overview

This AWS re:Invent 2025 session, presented by Lumen as an AWS Partner, explores how Lumen Defender Managed Rules enhance AWS Network Firewall by embedding real-time, backbone-level threat intelligence directly into cloud network defenses. The session targets both cloud strategists shaping security architecture and SOC analysts working on the front lines of detection and response, demonstrating how organizations can transform global network visibility into actionable protection for their AWS workloads.

## Turning Backbone Intelligence into Cloud Defense

### The Power of Visibility at Scale

Lumen anchors its security value proposition in the unmatched visibility provided by its global network. Through Black Lotus Labs, the company analyzes more than 200 billion NetFlow sessions every day, giving its researchers and threat hunters a vantage point that few organizations can replicate. This vast telemetry stream allows Black Lotus Labs to uncover botnets, track nation-state actors, and identify emerging command-and-control infrastructure long before traditional indicators surface in commercial threat feeds. The session emphasizes that this kind of backbone-level intelligence is fundamentally different from endpoint or perimeter-based detection, because it captures malicious behavior as it traverses the internet itself.

### Integration with AWS Network Firewall

Lumen translates this intelligence into Lumen Defender Managed Rules, which are delivered to customers through AWS Marketplace and applied directly to AWS Network Firewall. The integration is designed to operate with zero operational overhead: customers subscribe through Marketplace, and the managed rules continuously update as Black Lotus Labs validates new threats. By pushing curated, high-fidelity indicators into the native AWS firewall layer, Lumen enables organizations to block malicious traffic at the network boundary of their VPCs without requiring custom tooling, manual rule authoring, or complex integration work.

### Operational Impact for SOC Teams

A significant portion of the session focuses on the practical benefits for security operations. Because the managed rules are built from validated, behavior-driven intelligence rather than broad reputation lists, SOC teams see meaningfully fewer false positives and can dedicate their attention to genuine incidents. Faster, more confident blocking at the network layer also accelerates response times and reduces the dwell time of threats that might otherwise pivot deeper into cloud environments. The session positions this combination of curated intelligence and automated enforcement as a way to shift the SOC from reactive triage toward proactive defense, making backbone-level insight the first line of protection for cloud workloads running on AWS.
