# Color Grading and Finishing with FilmLight on AWS

## Overview of the Cloud-Based Grading Solution

The video demonstrates how professional color grading and finishing workflows are now available on Amazon Web Services through a partnership with FilmLight, a leading provider of color grading technology. By moving the traditionally on-premises Baselight grading system into the cloud, post-production professionals gain the ability to perform high-end color work without being tied to a specific physical facility. This shift enables remote collaboration, faster project turnaround, and scalable infrastructure that adapts to the demands of feature films, episodic content, and commercial production.

## Technology Stack and Signal Delivery

### Baselight in the Cloud

At the core of the demonstration is FilmLight's Baselight running as a cloud-based instance on AWS. Colorists interact with Baselight in the same way they would on-premises, retaining the full creative toolset and performance characteristics that the application is known for. Running the system in AWS removes the need for local high-performance hardware while still supporting the demanding processing requirements of professional color grading.

### AWS Cloud Digital Interface and MediaConnect

To preserve image fidelity from the cloud to the colorist's suite, the workflow combines AWS Cloud Digital Interface (CDI) with AWS Elemental MediaConnect. CDI provides uncompressed, low-latency video transport within the AWS network, while MediaConnect handles secure and reliable delivery of the signal out to the on-premises environment. Together, these services produce a visually lossless reference feed that drives a calibrated grading monitor in the colorist's room. This is essential for color-critical decisions, where any compression artifacts or color shifts would compromise the integrity of the creative process.

## Key Outcomes and Implications

The demonstration confirms that cloud infrastructure can now meet the strict technical requirements of color grading and finishing, a discipline historically considered too sensitive for remote or virtualized environments. By delivering a visually lossless signal to a reference monitor, AWS and FilmLight show that colorists can trust the cloud-based pipeline for final creative decisions. The combination of Baselight, CDI, and MediaConnect opens the door to more flexible production models, including remote colorists, distributed teams, and on-demand scaling of grading resources, ultimately reshaping how high-end finishing services can be delivered across the media and entertainment industry.
