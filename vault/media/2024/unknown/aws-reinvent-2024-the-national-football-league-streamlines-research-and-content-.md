# Streamlining NFL Research and Content Creation with Generative AI

## Overview of the Collaboration

The National Football League partnered with the AWS Generative AI Innovation Center to transform how production assistants research historical plays and create media content. As the first generative AI use case in NFL history, this initiative represents a significant milestone for the league, with the application slated for production deployment during the 2024 season. By leveraging Amazon Bedrock, the joint team built a solution that allows NFL staff to interact with vast archives of football data using everyday language rather than complex technical queries.

## The Business Challenge and Solution Approach

### Transforming Production Workflows

NFL production assistants traditionally faced significant friction when retrieving specific historical plays and statistics to support broadcasts, highlight reels, and editorial content. The process required navigating intricate data systems and APIs, slowing down content creation during time-sensitive production windows. The generative AI application addresses this challenge by accepting natural language queries such as "Show me all touchdown passes by John Smith with less than 2 minutes remaining," eliminating the technical barrier between researchers and the data they need.

### Routing Framework and API Integration

The core technical innovation lies in a routing framework that classifies the intent behind each user query and dynamically ingests the corresponding API schemas needed to fulfill the request. Once intent is identified, the system constructs the appropriate API endpoint call to retrieve the relevant data from NFL systems. This architectural approach allows the application to handle a wide variety of question types without requiring users to understand the underlying data model, while keeping the solution extensible as new APIs and data sources are introduced.

## Outcomes and Strategic Impact

By streamlining research and content creation, the application empowers production teams to surface compelling moments from NFL history in seconds rather than minutes or hours. The collaboration demonstrates how generative AI, grounded in well-structured enterprise APIs and orchestrated through Amazon Bedrock, can deliver tangible productivity gains for media-intensive organizations. For the NFL, this initial success establishes a foundation for further generative AI adoption across the league, while showcasing a reusable pattern—intent classification combined with dynamic API schema routing—that other enterprises can apply to their own knowledge retrieval challenges.
