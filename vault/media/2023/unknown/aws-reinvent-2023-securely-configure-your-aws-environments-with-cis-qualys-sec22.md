# Securely Configuring AWS Environments with CIS and Qualys

## Overview of the Session

This AWS re:Invent 2023 lightning talk, delivered in partnership with the Center for Internet Security (CIS), addresses one of the most persistent challenges organizations face during cloud adoption: maintaining workload security and regulatory compliance. The session highlights how the collaboration between CIS and Qualys equips AWS customers with practical tools to harden their environments against misconfiguration risks, which remain among the leading causes of cloud security incidents. By focusing on industry-recognized benchmarks and automated assessment capabilities, the presentation demonstrates a path toward consistent, scalable cloud security governance.

## Securing AWS Workloads with Industry Benchmarks

### The Role of the CIS AWS Foundations Benchmark

At the core of the discussion is the CIS AWS Foundations Benchmark, a globally recognized set of prescriptive configuration recommendations developed through community consensus. The benchmark covers critical AWS services such as AWS Identity and Access Management (IAM), Amazon S3, and Amazon EC2, providing organizations with a vetted baseline to evaluate the security posture of their AWS accounts. Adopting these guidelines helps customers align with broader compliance frameworks and reduce the attack surface created by default or overly permissive configurations.

### How Qualys TotalCloud Operationalizes the Benchmark

Qualys TotalCloud builds on this foundation by automating the assessment and continuous monitoring of AWS environments against the CIS benchmark. Rather than relying on manual audits, customers gain visibility into configuration drift and policy violations in near real time, enabling faster remediation. The integration streamlines compliance reporting and supports security teams in scaling oversight across multiple AWS accounts and services without sacrificing rigor.

### Partnering with CIS for Broader Security Outcomes

The session also encourages organizations to engage directly with CIS, whether by adopting benchmarks, contributing to community-driven security guidance, or leveraging CIS Hardened Images available through AWS. This collaborative model allows companies to extend the same proven, consensus-based security practices into their own products and customer offerings, reinforcing a shared-responsibility approach to cloud security.

## Key Takeaways

The central message is that securing AWS environments does not require organizations to start from scratch. By combining the authoritative guidance of the CIS AWS Foundations Benchmark with the automation and continuous assessment capabilities of Qualys TotalCloud, AWS customers can establish a repeatable, audit-ready security baseline. The partnership between CIS, Qualys, and AWS exemplifies how vendors, standards bodies, and cloud providers can jointly simplify compliance and strengthen security outcomes for customers at any stage of cloud maturity.
